## :rocket: Installation
```bash
# To clone the repository
$ git clone https://github.com/Uriel96/webtoo-frontend.git webtoo-frontend

# To install all dependencies (inside webtoo-frontend project)
$ npm install
OR
$ yarn install

# To run the app
$ npm run serve
OR
$ yarn serve
```
